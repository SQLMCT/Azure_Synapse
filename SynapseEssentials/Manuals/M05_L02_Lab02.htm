Exercise 1: Resource Monitoring with Azure Monitor

In this exercise you will learn how to use metrics to determine the load on your Synapse SQL pool.

Tasks

1.        Connect to the Azure Portal

Open Internet Explorer and connect to the Azure portal at https://portal.azure.com

2.        Go to Resource Groups

On the Home page search and select Resource Groups


     

Select the Resource Group that contains your Synapse SQL pool

 

 

 

3.        Select your Synapse SQL pool, created in module 1

Select the Synapse SQL pool.
The following image might differ from your list of SQL pool.

 

4.        Select Metrics

Click on Metrics under Monitoring.

 

5.        Monitor CPU Percentage Metric

Click on the “Metric” dropdown and select “CPU Percentage”.



This will display the CPU percentage (Avg) chart for the last 24 hours.

6.        Monitor Data IO Percentage Metric

Click on “Add Metric” to add a new metric to monitor.



In the “Metric” dropdown, click on “Data IO Percentage”.

7.        Adjusting the Time Range for your Chart

Click on the time range dropdown to adjust the time range for your chart.



Change the time range from “Last 24 hours” to “Last hour” and click “Apply”.



Observe the change to data range in your chart.

 

 

8.        Adding Additional Charts for monitoring

Click on “+ New chart” to add a new chart.



This will create add a new chart, where you can select additional metrics to monitor.
